# Castlebound-RiseOfTheKnight

Castlebound rise of the knight is een 2D platformer waarbij je levels moet halen als een ridder. 
Deze ridder heeft meerdere abilities zoals:

- Jumping
- Walljump
- Doublejump
- Charged jump
- Attack (swing sword)
- Block met schild

Enemies:
- Ridder
- Spikehead (still has some issues)

Traps:
- Spikes (floor & wall spikes)
- Arrowtrap
- Flametrap (flamethrower)
- Speartrap
- Saw (can move)

Collectables:
- Hearts
- Checkpoints
- Finish flag

Er is momenteel een level uitgewerkt dit level dient als een samenvatting van alle items in het spel en is niet het startlevel voor de uiteindelijke game.
